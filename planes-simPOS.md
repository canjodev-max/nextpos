# PLANES Y PRECIOS — simPOS
### Sistema de Punto de Venta con Facturación Electrónica Paraguay

---

## 🟡 PLAN INICIAL
### Gs. 250.000 / mes — precio fijo

Ideal para negocios que están comenzando y necesitan un sistema de ventas confiable y profesional.

### ✅ Funciones incluidas

**Punto de Venta (POS)**
- Terminal de venta táctil, rápida y fácil de usar
- Búsqueda de productos por nombre, código interno o código de barras
- Venta por unidad y por peso (productos a granel)
- Aplicación de descuentos por producto o por venta
- Múltiples métodos de pago: efectivo, tarjeta, QR, transferencia
- Cálculo automático de vuelto
- Registro de ventas a crédito (cuenta corriente del cliente)
- Devoluciones y anulaciones de ventas

**Gestión de Inventario**
- Alta, edición y baja de productos
- Código interno y código de barras por producto
- Imagen del producto
- Precio de venta y precio de costo
- Control de stock con alertas de stock mínimo
- Productos sin control de stock (venta directa)
- Categorías de productos
- Descuentos por producto (porcentaje)
- Precio mayorista y cantidad mínima para mayoreo
- Marcado de productos como prioritarios
- Registro de mermas y desperdicios
- Reposición de stock (ingreso de mercadería)
- Kardex de movimientos por producto (historial completo)
- Vista de productos faltantes (bajo stock)
- Vista de productos en oferta

**Gestión de Caja**
- Apertura de caja con monto inicial
- Registro de ingresos y egresos manuales
- Resumen en tiempo real: efectivo, tarjeta, QR, transferencia
- Cierre de caja con arqueo y diferencias
- Historial de sesiones de caja anteriores
- Registro de auditoría de cada movimiento

**Gestión de Clientes**
- Base de datos de clientes con cédula/RUC
- Datos de contacto: teléfono, email, fecha de nacimiento
- Cuenta corriente: saldo deudor por cliente
- Historial de compras por cliente
- Registro y seguimiento de deudas
- Pagos parciales de deudas

**Usuarios y Roles**
- Múltiples usuarios por negocio
- Roles predefinidos: ADMIN, SUPERVISOR, CAJERO
- Permisos configurables por rol:
  - Ver inventario
  - Crear / editar / eliminar productos
  - Acceso al POS
  - Abrir y cerrar caja
  - Gestionar usuarios
  - Ver reportes

**Dashboard y Reportes**
- Panel principal con ventas del día, stock total y alertas
- Transacciones recientes
- Alertas críticas de stock y vencimientos
- Resumen de caja en tiempo real

**General**
- Acceso desde cualquier dispositivo con navegador (PC, tablet, celular)
- Múltiples usuarios simultáneos
- Notificaciones del sistema
- Soporte por WhatsApp en horario comercial (lunes a viernes 8:00–18:00)

### ❌ No incluye
- Facturación electrónica (e-Kuatia / SIFEN)
- Soporte fuera de horario comercial

## 🔵 PLAN COMPLETO
### Gs. 300.000 / mes + facturas adicionales

Para negocios que necesitan el sistema completo con facturación electrónica habilitada ante la SET.

### ✅ Todo lo del Plan Inicial, más:

**Facturación Electrónica (e-Kuatia / SIFEN)**
- Emisión de facturas electrónicas con validez fiscal ante la SET
- Emisión de notas de crédito y notas de débito electrónicas
- Generación del KuDE (representación imprimible de la factura electrónica)
- Envío automático de documentos a SIFEN (SET Paraguay)
- Consulta de estado de documentos electrónicos
- Cancelación de documentos electrónicos
- Soporte para régimen normal, pequeño contribuyente y microempresa
- Configuración de timbrado, establecimiento y punto de expedición
- Generación de QR en el KuDE con código de seguridad (CSC)
- Hasta **150 facturas electrónicas por mes** incluidas

**Soporte**
- Soporte por WhatsApp en horario comercial (lunes a viernes 8:00–18:00)

### 📊 Facturas adicionales

Cada bloque de 20 facturas extra por encima de las 150 incluidas tiene un costo adicional de **Gs. 20.000**.

| Facturas emitidas en el mes | Costo mensual total |
|---|---|
| Hasta 150 | Gs. 300.000 |
| 151 – 170 | Gs. 320.000 |
| 171 – 190 | Gs. 340.000 |
| 191 – 210 | Gs. 360.000 |
| 211 – 230 | Gs. 380.000 |
| 231 – 250 | Gs. 400.000 |
| Cada 20 más | +Gs. 20.000 |

### ❌ No incluye
- Soporte fuera de horario comercial
- Múltiples sucursales

## 🟣 PLAN ENTERPRISE
### 1% de los ingresos netos mensuales

Para empresas con alto volumen de operaciones que necesitan el sistema completo, facturación ilimitada y soporte prioritario las 24 horas.

### ✅ Todo lo del Plan Completo, más:

**Facturación Electrónica Ilimitada**
- Sin tope de facturas electrónicas por mes
- Facturación ilimitada sin costos adicionales

**Múltiples Sucursales**
- Gestión de varios establecimientos desde una sola cuenta
- Cada sucursal con su propio punto de expedición y caja
- Reportes consolidados por sucursal o globales

**Soporte Prioritario 24/7**
- Atención por WhatsApp, llamada telefónica y videollamada
- Respuesta garantizada en menos de 2 horas
- Soporte los 7 días de la semana, incluyendo feriados
- Visitas técnicas presenciales incluidas (según zona)

**Implementación y Capacitación**
- Configuración completa del sistema incluida
- Capacitación presencial o remota al equipo del negocio
- Acompañamiento en el proceso de habilitación ante la SET

**Actualizaciones y Mantenimiento**
- Todas las actualizaciones del sistema incluidas sin costo adicional
- Respaldo automático de datos
- Recuperación de datos ante cualquier incidente

**Reportes Avanzados**
- Reportes personalizados según las necesidades del negocio
- Exportación de datos en Excel / PDF

### 💰 Precio
El **1%** se calcula sobre los **ingresos netos declarados del mes anterior**.

*Ejemplo: si el negocio facturó Gs. 60.000.000 netos en el mes, el costo del siguiente mes es Gs. 600.000.*

> El cliente informa sus ingresos netos mensualmente. simPOS puede verificar mediante los documentos electrónicos emitidos a través del sistema.

## ⚫ PLAN PROPIETARIO (LICENCIA LOCAL)
### Gs. 20.000.000 — pago único

Para empresas que prefieren ser **dueñas del sistema**, sin mensualidades, funcionando completamente en su propia infraestructura.

### ✅ Todo lo del Plan Enterprise, más:

**Propiedad del Sistema**
- Licencia perpetua del sistema simPOS
- El sistema pasa a ser propiedad total del cliente
- Sin pagos mensuales ni porcentajes sobre ventas
- Sin dependencia de servidores externos

**Instalación Local**
- Instalación en servidor o PC del cliente
- El sistema funciona en red local (LAN) sin necesidad de internet
- Acceso desde todas las computadoras y dispositivos del negocio en la red local
- Configuración completa de la base de datos local

**Facturación Electrónica Ilimitada**
- Sin tope de facturas electrónicas
- Facturación ilimitada de por vida sin costo adicional
- Todos los tipos de documentos: facturas, notas de crédito, notas de débito, remisiones, autofacturas

**Soporte Completo por 3 Meses**
- Soporte 24/7 durante los primeros 3 meses desde la instalación
- Visitas técnicas presenciales incluidas en los 3 meses
- Capacitación completa al equipo del negocio
- Acompañamiento total en la habilitación ante la SET (SIFEN)
- Resolución de cualquier incidente sin costo adicional durante el período

**Después de los 3 meses de soporte incluido:**
- Soporte extendido disponible por contrato separado
- Actualizaciones disponibles como servicio adicional

### 🖥 Requisitos mínimos del servidor del cliente
- PC o servidor dedicado con Windows 10/11 o Linux
- 8 GB de RAM mínimo (16 GB recomendado)
- 100 GB de almacenamiento disponible
- Red local (LAN) para acceso desde múltiples dispositivos
- Conexión a internet solo para envío de documentos a SIFEN

## 🖥 HARDWARE — Servicio Opcional

El hardware se cotiza y prepara **por separado**, según las necesidades de cada negocio. simPOS puede proveer, configurar e instalar todo el equipamiento listo para usar con el sistema.

> El precio de cada equipo se tasa individualmente al momento de la consulta.

### Equipos disponibles

**Terminal / Computadora POS**
- PC de escritorio o All-in-One táctil
- Tablet o pantalla táctil para el cajero
- Mini PC para instalación en mostrador

**Impresora de Tickets / Facturas**
- Impresora térmica de 80mm (tickets de venta)
- Impresora de facturas A4 (para KuDE / factura electrónica)

**Lector de Código de Barras**
- Lector USB de mano
- Lector inalámbrico Bluetooth

**Cajón de Dinero**
- Cajón portamonedas con apertura automática conectado a la impresora

**Pantalla para el Cliente**
- Display secundario que muestra el total al comprador

**Router / Red Local**
- Switch y cableado de red para múltiples terminales
- Router WiFi para acceso inalámbrico

**Servidor Local** *(requerido para Plan Propietario)*
- PC o servidor dedicado configurado y listo para el sistema

### ¿Cómo funciona?

1. El cliente indica qué equipos necesita o se hace una visita para evaluar el local
2. Se cotiza cada equipo por separado según disponibilidad y marca
3. Se prepara el hardware con el sistema ya instalado y configurado
4. Se entrega listo para usar, con capacitación incluida

> **Nota:** La compra de hardware es completamente opcional e independiente del plan de software contratado. El cliente puede adquirir su propio hardware siempre que cumpla los requisitos mínimos del sistema.
